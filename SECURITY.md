# Security

If you believe you have found a security vulnerability in this project, please report it responsibly.

- **Do not** open a public GitHub issue for undisclosed security problems.
- Email or open a **private** security advisory on GitHub (repository **Security** tab → **Report a vulnerability**), if enabled.

Include enough detail to reproduce or understand the impact (version, affected code paths, proof of concept if safe to share).

We will acknowledge receipt and work toward a fix and disclosure timeline in line with common coordinated disclosure practice.
